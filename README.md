# Apple E-commerce Performance Analytics

##  Project Overview

This project analyzes Apple product pricing and e-commerce performance across Amazon and Flipkart.

The analysis explores product categories, product conditions, pricing, discounts, ratings, reviews, stock availability, and sale events to identify meaningful business insights.

##  Tools Used

- Python
- Pandas
- Matplotlib
- SQL
- Power BI

##  Key Analysis Areas

- Platform performance comparison
- Product category analysis
- New vs Renewed product comparison
- Discount analysis
- Sale event performance
- Stock status analysis
- Rating and review analysis
- Pricing analysis

## Project Workflow

1. Data cleaning and exploration using Python
2. Exploratory data analysis using Pandas
3. Business analysis using SQL
4. Dashboard creation using Power BI
5. Deriving actionable business insights

##  Files Included

- Python analysis notebook
- SQL analysis queries
- Power BI dashboard

##  Key Insights

- Renewed products offered higher average discounts than new products.
- Major sale events had significantly higher discounts than regular periods.
- Renewed products had lower average ratings compared with new products.
- Discount levels varied significantly across product categories and stock statuses.

##  Objective

To analyze e-commerce pricing and product performance data and provide data-driven insights to support pricing, inventory, and promotional decisions.
